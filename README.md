# Élegante Parfums

**Élegante Parfums** is a luxurious perfume brand website concept that blends timeless beauty with modern web design.  
Built with **HTML**, **CSS**, and **JavaScript**, it captures the essence of sophistication through smooth animations, transparent visuals, and high-quality imagery creating an immersive shopping and brand experience.

---

## Overview
Élegante Parfums represents the art of digital elegance.  
From the first glance, visitors are greeted by a stunning transparent hero section featuring the signature perfume bottle and golden highlights that emphasize luxury and detail.

This project demonstrates both **frontend craftsmanship** and **brand storytelling** through minimalist yet expressive design.

---

## Features
-  Transparent hero section with realistic 3D-inspired visuals  
-  Subtle motion effects  
-  Fully responsive design for all devices  
-  Smooth transitions and interactive hover animations  
-  Optimized image performance and visual clarity  

---

## Tech Stack
- **HTML5** – structure  
- **CSS3** – styling and responsiveness  
- **JavaScript (Vanilla)** – interactions and motion effects
